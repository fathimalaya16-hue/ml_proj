# Diabetes Prediction System

## Overview
The Diabetes Prediction System is a Machine Learning project that predicts diabetes risk using health-related data such as glucose level, BMI, insulin, age, and pregnancies.

This project is built using:
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

---

## Features
- Predict diabetes risk instantly
- User-friendly Streamlit web interface
- Machine Learning based prediction
- Health data analysis
- Fast and simple UI

---

## Project Files

- `diabetes.py` → Main Streamlit application
- `diabetes.ipynb` → Model training notebook
- `diabetes2.csv` → Dataset
- `diabetes_model.pkl` → Trained ML model
- `diabetes_scaler.pkl` → Feature scaler
- `diabetes_img.jpg` → Homepage image
